Argument	1.
Argument	Section 144 of the Code of Civil Procedure cannot, in terms apply as no decree was varied or reversed but only an order under Order XXI, Rule 90, refusing to set aside a sale in execution, was reversed by <ORG>. '
Argument	2.
Argument	Assuming that Section 151 of the Code of Civil Procedure allows an order for restitution in appropriate cases even though it does not fall under Section 144 of the Code of Civil Procedure, such an order cannot be made on the analogy of Section 144 unless the auction-purchaser was a party before <ORG> which set aside the sale in the proceedings instituted for setting it aside.
Argument	3.
Argument	The mere fact that the decree-holder was a party to those proceedings will not suffice as the <ORG> auction-purchaser is not the representative of the decree-holder.
Argument	<NAME> v. <NAME> (1907) I.L.R., 30 Mad., 507 which held otherwise has been disapproved of in <CITATION_891538> (1911) I.L.R., 34 Mad., 417 to both of which decisions the present  learned Chief Justice was a party.
Argument	4.
Argument	The <NAME> says, that the 'auction-purchaser need not at all be a party to any proceedings for confirmation of or setting aside the sale.'
Fact	The petition for restitution does not expressly state that the <ORG> auction-purchaser was a party to the proceedings taken to have the auction sale set aside.
RulingByLowerCourt	No records have been produced before us to establish that fact.
RulingByLowerCourt	We therefore cannot hold that Section 151 empowers the <ORG> to pass an order for restitution against a person not shown to be a party to and so bound by the order of the Appellate <ORG> setting aside the sale.
RulingByLowerCourt	5.
RulingByLowerCourt	The appeal is dismissed with costs.
