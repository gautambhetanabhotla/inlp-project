RatioOfTheDecision	The effective return for the purposes of assessment is thus the return which is ultimately filed by an assessee on the basis of which he wants his income to be assessed.
RatioOfTheDecision	Sub-section (5) is a part of Section 139 .
RatioOfTheDecision	The statute itself provides for the filing of a revised return.
RatioOfTheDecision	Section 143 does not specifically deal with the return filed either under Sub-section (1) or Sub-section (2) or Sub-section (4) of Section 139 .
RatioOfTheDecision	It deals generally with the return filed under Section 139 .
RatioOfTheDecision	It is true that Sub-section (5) specifically mentions Sub-section (1) or Sub-section (2) but does not mention the case of a return filed under Sub-section (4) of Section 132 .
RatioOfTheDecision	However, the true purport of a return filed under Sub-sec tion (5) is that it substitutes the original return filed and Sub-section (4) specifically gives the assessee a right to file a return at any time before the assessment.
RatioOfTheDecision	Hence, where a voluntary return has been filed under Section 139(4) , a revised return can be filed in respect of it.
