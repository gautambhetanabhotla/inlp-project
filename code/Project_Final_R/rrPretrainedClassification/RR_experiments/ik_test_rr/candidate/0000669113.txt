RatioOfTheDecision	A step taken by the judgment-creditor himself is not, as is pointed out in one of the cases to which we have referred, sufficient.
RatioOfTheDecision	2.
RatioOfTheDecision	An application for leave to bid is not, as far as we can see, in any way a step in aid of execution, It is merely an order made to remove the restriction which the Code of Civil Procedure has placed upon the action of the judgment-creditor.
RulingByPresentCourt	The sole object of it is to remove that restriction and not to enforce the decree.
RulingByPresentCourt	For these reasons we dismiss the appeal with costs.
